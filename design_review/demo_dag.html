Interaction Guide

Add Node

  • Double-click canvas → Add node
  • Drag from plugin bar → Drop on canvas
  • Right-click canvas → Add Node menu

Select & Move

  • Click node → Select
  • Drag node → Move
  • Shift+Click → Multi-select
  • Click canvas → Deselect all

Connect

  • Drag from ■ output port → ■ input port

Right-Click

  • Node → Copy / Duplicate / Disable / Delete
  • Canvas → Add Node menu

Keyboard

  • Del → Delete selected
  • Ctrl+D → Duplicate
  • E → Disable/Enable
  • Ctrl+Z/Y → Undo/Redo
