# GUI 设计审查 — #02 FilmstripView 胶片条

---

## 一、界面设计（UI Design）

### 1.1 设计文档定义的界面

ARCHITECTURE.md 第 8.1 节将胶片条定义为左栏顶部区域：

```
胶片条
（缩略图列表 + 分组树）
```

**设计预期：**
- 缩略图网格/列表：显示已导入图像的缩略图
- **分组树**：按 ISO、GPS 聚类、时间间隔自动分组，树状结构显示
- 与批量进度共享左栏（胶片条在上，批量进度在下）

**交互预期：**
- 点击缩略图 → 选中 → 在预览区显示
- 多选 → 发送到批量处理
- 右键上下文菜单 → 文件操作

### 1.2 实际实现的界面

4. 筛选栏 Row 3：TextBox（实时筛选）+ ComboBox（格式过滤）

## 二、功能设计（Functional Design）

### 2.1 职责边界

FilmstripViewModel 负责：
- 图像导入（文件对话框 → 加载 → 生成缩略图）
- 图像管理（添加/移除/清除/多选）
- 排序（按名称/大小/格式）
- 筛选（按文本/格式）
- 向 Batch 发送图像集合
- 文件系统集成（打开资源管理器/复制路径）

### 2.2 数据流向

```
用户导入图像 → FilmstripVM.AddImages()
  → 生成缩略图（SemaphoreSlim 并发限制 4）
  → FilteredImages 更新
  → ListBox 显示

用户选择图像 → SelectedImage → PreviewVM (通过 MainVM 中介)
  → 预览区加载图像

用户点击 "To Batch" → SendToBatch()
  → BatchVM.BatchQueue 添加 SelectedImages
```

### 2.3 用户操作流程

| 步骤 | 操作 | 系统响应 | 状态变化 |
|------|------|---------|---------|
| 1 | 点击 Import | 打开文件对话框 | 空态 → 加载态（Async） |
| 2 | 选择图像文件 | 生成缩略图 | 加载态 → 内容态 |
| 3 | 点击缩略图 | 选中 → 加载到 Preview | 选中状态 |
| 4 | Ctrl+Click 多选 | 多选集合更新 | 多选状态 |
| 5 | 右键 → 发送到 Batch | 图像加入批量队列 | Batch 侧更新 |
| 6 | 在筛选框输入文字 | 实时 FilteredImages 过滤 | 列表动态收缩 |
| 7 | 选择格式过滤 | 按格式筛选 | 列表仅显示匹配格式 |
| 8 | 选择排序方式 | 列表重新排序 | 排列顺序变化 |

### 2.4 与其他组件的协作

| 协作方 | 关系 | 实现 |
|-------|------|------|
| PreviewViewModel | 选中图像 → 加载预览 | MainViewModel 中介事件 |
| BatchViewModel | "To Batch" → 添加图像 | SendToBatch() 方法调用 |
| ImageService | 生成缩略图 | ImageService.GetThumbnailAsync |

## 三、实现程度（Implementation Assessment）

### 3.1 已实现

- ✅ 导入图像（文件对话框，多选支持）
- ✅ 缩略图生成（异步，SemaphoreSlim=4 并发限制）
- ✅ 排序（名称/大小/格式）
- ✅ 文本筛选（PropertyChanged 实时触发）
- ✅ 格式筛选（ComboBox）
- ✅ 右键上下文菜单（全选/反选/清除选择/移除/发送/打开/复制路径）
- ✅ 多选（Extended 模式，代码后置同步 SelectedImages）
- ✅ 缩略图 S/M/L 切换（80/120/180px 加载尺寸）
- ✅ 清除图像
- ✅ 发送到 Batch

### 3.2 未实现

| 缺失项 | 设计依据 | 严重度 |
|-------|---------|:-----:|
| ❌ **分组树（按ISO/GPS/时间分组）** | 设计文档明确写"缩略图列表 + 分组树" | 🔴 严重 |
| ❌ **空态占位提示** | 基本 UX 规范 | 📝 体验缺失 |
| ❌ **导入中加载动画** | 基本 UX 规范 | 📝 体验缺失 |
| ❌ **筛选无结果提示** | 基本 UX 规范 | 📝 体验缺失 |
| ❌ **拖放导入（从资源管理器）** | 通用桌面应用预期 | 📝 交互缺失 |
| ❌ **分组树展开/折叠交互** | 设计文档要求的树状结构 | 🔴 严重（同分组树） |

### 3.3 关键交互问题

1. **筛选时滚动位置丢失** — `FilteredImages` 被替换为全新 ObservableCollection，重置 ListBox 滚动位置
2. **筛选/排序清除选中状态** — `ApplyFilter()` 内部调用 `SelectedImages.Clear()`
3. **缩略图取消无处理** — 缩略图生成中的 `CancellationToken` 无法阻止已在运行的任务
4. **无"全选"快捷键**（Ctrl+A） — 需通过右键菜单

### 3.4 设计偏离汇总

| 偏离项 | 设计 | 实际 | 影响评估 |
|--------|------|------|---------|
| 分组树 | 胶片条包含缩略图列表 + 分组树 | 仅有缩略图列表 | 🔴 重大缺失 |
| 批量进度位置 | 胶片条下方 | 在右侧 BatchView | ⚪ 布局决策变更 |
