🔌 Connection Rules

Input port (⊡ green square)
Accepts 1 incoming connection. Hover shows tooltip "← input".
Output port (⊡ blue square)
Can fan out to multiple nodes.
Creating a connection
Drag from output port → drop on input port. Dashed preview line follows cursor.
Removing a connection
Right-click port → "Disconnect" · or · select edge (hover) → Del key.
Cycle prevention
Red flash + "Connection would create a cycle" toast if detected.
Fan-out
One output → multiple inputs allowed (no warning needed).

🎯 Interaction Summary

Add node
① Drag from plugin panel · ② Double-click plugin card · ③ Right-click canvas → Add Node
Select
Click node. Shift+Click for multi. Click+drag canvas to box-select.
Delete node
① Del key · ② Right-click → Delete · ③ Right panel 🗑 button
Move node
Click+drag. Holding Shift locks to axis.
Connect
Drag from output port to input port.
Minimap
Click minimap to jump viewport. Drag viewport to pan.
Canvas zoom
Scroll wheel (centered on cursor). Buttons ±/Fit. Ctrl+0 = Reset.
