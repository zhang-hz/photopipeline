## 一、界面设计（UI Design）

### 1.1 设计文档定义的界面

ARCHITECTURE.md 第 8.1 节将预览区定义为中心区域：

```
┌──────────────────────────────┐
│         预览区                │
│   处理前 │ 处理后             │
│   分割 / 对比                │
│                              │
├──────────────────────────────┤
│       管线总览               │
│     迷你 DAG + 状态          │
├──────────────────────────────┤
│         信息栏               │
└──────────────────────────────┘
```

**设计预期（3 个子区域）：**
1. **预览区**（顶）：处理前/处理后分屏对比，支持分割/对比模式切换
2. **管线总览**（中）：迷你 DAG 图的只读概览视图 + 节点状态
3. **信息栏**（底）：像素信息、状态消息等

### 1.2 实际实现的界面

PreviewView.xaml 实现：

```
┌────────────────────────────────────────┐
│ 🔍+ 🔍- [Fit] [1:1] │ 125% │ Split │Export│ ← 工具栏
├────────────────────────────────────────┤
│                                        │
│         SkiaPreviewCanvas              │
│    ┌──────────────┬──────────────┐     │
│    │    Before    │    After     │     │
│    │              │              │     │
│    │   (拖动分割线)              │     │
│    │              │              │     │
│    │              │              │     │
│    └──────────────┴──────────────┘     │
│                                        │
├────────────────────────────────────────┤
│  X: 1200 Y: 800  R:128 G:64 B:32  │ ⟳ Loading...│
└────────────────────────────────────────┘
```

**实际的三行布局：**
1. 工具栏 Row 0：Border 背景，Zoom+ / Zoom- / Fit / 1:1 / Separator / ZoomLevel% / Separator / Split / Export
2. 画布 Row 1：SkiaPreviewCanvas（SkiaSharp 硬件加速渲染），绑定 7 个依赖属性
3. 状态栏 Row 2：左对齐 PixelInfo + 右对齐 Loading 指示器（ProgressRing + "Loading..."）

## 二、功能设计（Functional Design）

### 2.1 职责边界

PreviewViewModel + SkiaPreviewCanvas 负责：
- 加载图像（通过 gRPC ImageService.DecodeAsync）
- 处理管线（通过 gRPC PipelineService.ExecuteAsync）
- Before/After 分屏对比显示
- 缩放/平移/适配窗口
- 图像导出
- 悬停像素信息采样

### 2.2 数据流向

```
FilmstripVM 选择图像 → MainVM 中介
  → PreviewVM.LoadImageAsync(path)
  → BeforeBitmap = 解码后的原始图像

PipelineEditorVM 执行管线 → MainVM 中介
  → PreviewVM.ProcessPipelineAsync()
  → AfterBitmap = 管线处理后的图像

用户交互（滚轮/拖动/点击）
  → SkiaPreviewCanvas 内部状态
  → ZoomLevel/PanOffset 双向绑定 → PreviewVM
```

### 2.3 用户操作流程

| 步骤 | 操作 | 系统响应 | 状态变化 |
|------|------|---------|---------|
| 1 | 在胶片条选择图像 | 预览区加载原始图像 | 空态 → Before 显示 |
| 2 | 鼠标滚轮 | 画布以鼠标位置为中心缩放 | 缩放变化 |
| 3 | 鼠标拖动 | 画布平移 | 平移变化 |
| 4 | 点击 Split | 切换分屏模式 | Before/After 并排 |
| 5 | 拖拽分割线 | 分屏比例变化 | 分割位置变化 |
| 6 | 点击 Fit | 自动适配窗口 | 缩放/平移重置 |
| 7 | 执行管线 | After 图像更新 | After 侧显示 |
| 8 | 点击 Export | 保存对话框 → 编码导出 | 导出流程 |
| 9 | 悬停像素 | 像素信息更新 | 状态栏信息变化 |

### 2.4 与其他组件的协作

| 协作方 | 关系 | 实现 |
|-------|------|------|
| FilmstripViewModel | 图像选择 → 预览加载 | MainVM 中介事件 |
| PipelineEditorViewModel | 管线执行 → 预览更新 | MainVM 中介事件 |
| ImageService | 解码图像 | gRPC DecodeAsync |
| PipelineService | 执行管线获取 After | gRPC ExecuteAsync |
